Sharing SVG's 2016-12-14

A couple of years ago, I was frustrated that no social network silo treated SVG as an image format, insisting it was made into a bitmap before sharing. So I made svgshare.com (formerly known as svgur.com ).

This makes it easy to upload SVGs and share a link, and includes the duelling facebook and twitter metadata and links to a PNG version so when you share a link on those sites, you see a PNG preview.

A reason I'm given for SVG wariness is that they are HTML documents that you can embed <script> tags in, causing bad things to happen. While this is true, putting SVG in an img tag means that scripts don't run, and svgshare was built around this from the beginning. The only place I embed an SVG directly in the page is as part of the upload preview process, so the people who regularly upload SVGs with script in end up pwning themsleves, rather than my users.

However, as these are getting more common, I added some code to filter out SVGs with scripts in. I used html5lib to parse them and strip out the script.
